# smart-pot
노인의 건강 및 정서케어를 위한 스마트 화분

## 개요
- Raspberry Pi 5 + ESP32 기반 스마트 화분
- 기능: 센서 데이터 수집하여 정서적으로 안정된 환경 유도, 표정 인식 후 정서 상태 기록

## 기술 스택
- Hardware: Raspberry Pi 5, ESP32, BH1750, CCS811, YL-83
- Software: Python, C++, OpenCV, Flask
- Communication: UART, I2C
